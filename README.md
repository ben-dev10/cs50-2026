# CS50 2026

This repo contains source files to problems, lectures and general practice files and interactive notebooks for the CS50 2026 course on EdX by Harvard.


## Note
This repo was developed on WSL:Ubuntu system. You would have to install C and Python on your system first.

To run the Jupyter notebooks:
- Install Conda or Miniconda
- Create a new environment for py or c/c++ kernels
- Install a kernel of your choosing. For this project I used the following kernels:
    - the standard pip py kernel
    - xeus-cling for C++
    - jupyter-c-kernel for C
